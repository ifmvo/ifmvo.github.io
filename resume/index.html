实习经历

黑龙江大米网电子商务有限公司 （ 2016.12 ~ 至今 ）

黑龙江大米网交易平台

黑龙江大米网交易平台是 B2B 电商类 App。作为项目的核心程序员我不但完成了项目 Android 端界面和功能开发，也负责移动端界面设计，更使用了组件化架构项目，通过 MVP 模式使项目逻辑清晰高效运行，还提出 Git Flow 流式版本控制。

在 Android 端 App 上线后我将平均占用内存从 40M 优化到 25M，代码量从 15w 降到 10w 行，Apk 包大小从 12M 缩小到 7M，我解决图片闪烁，列表卡顿等问题使领导对我刮目相看。

在 通过 ICE 网络请求时，通常使用 Callback 回调的传统方式做请求返回处理，而我通过 RxJava 将 ICE、Loading、异常处理、线程调度完美封装，实现最简单、方便的网络请求调用。

哈尔滨时光慢递有限责任公司（ 2016.4 ~ 2016.11 ）

拾忆

拾忆是移动端图像摄影、PC制作并打印相册相结合的平台 ，我负责项目 Android 端所有功能的实现，在 App 首页使用高性能大图片列表展示，动态页面使用多类型列表，采用粒子爆炸实现星空效果并使用属性动画实现旋涡、图标呼吸效果，商城采用 WebView + html5 实现完美兼容 JS 和 CSS 样式。

升任至拾忆项目负责人后，我带领一个4人小组维护着每天 1000 在线量的平台。
